# 個人網站｜王奕崴
相較於制式履歷，我嘗試以網頁 —— 這個更自由的載體，去收納與統整我的個人資訊，包括背景、技能、作品集、學經歷等。希望藉由簡潔活潑的視覺風格，提供瀏覽者更舒服與直觀的體驗。

## DEMO
[前往頁面](https://wangyiwei0108.github.io/personal-website/)

## 使用技術
- HTML / CSS / JS
- SASS
- React

## 實現功能

- 當滑鼠游標移動時，呈現尾巴狀的動態視覺
- 背景顏色與字體顏色，隨著頁面滾動而變換
- Navbar：根據畫面尺寸，隱藏或顯示漢堡選單
- RWD：瀏覽器畫面支援手機、平板、電腦等裝置
